# Ícones do PWA - Cliente

## Ícones Necessários

### Ícones Principais
- `customer-192.png` - 192x192px (obrigatório para PWA)
- `customer-512.png` - 512x512px (obrigatório para PWA)

### Ícones de Atalhos
- `orders.png` - 96x96px (atalho para Pedidos)
- `boat.png` - 96x96px (atalho para Embarcações)

## Especificações de Design

### Tema de Cores
- Cor Principal: `#10b981` (Verde)
- Cor de Fundo: `#ffffff` (Branco)
- Estilo: Amigável, moderno, acessível

### Elementos Visuais
- Ícone: Âncora (anchor) ou barco estilizado
- Background: Degradê verde ou sólido
- Bordas: Arredondadas (radius: 20-25%)
- Shadow: Sutil, para profundidade

## Como Gerar os Ícones

### Opção 1: Usando Figma/Adobe Illustrator
1. Criar artboard 512x512px
2. Desenhar ícone com padding de 64px (margens)
3. Exportar em PNG com fundo transparente ou sólido
4. Redimensionar para 192x192px

### Opção 2: Usando Ferramentas Online
- https://realfavicongenerator.net/
- https://www.pwabuilder.com/imageGenerator
- https://favicon.io/

### Opção 3: Usando IA (ChatGPT/Midjourney)
Prompt sugerido:
```
Create a modern, friendly app icon for a maritime e-commerce customer app.
Style: Flat design, approachable, friendly
Colors: Green (#10b981) and white
Icon: Anchor or stylized boat
Size: 512x512px, with rounded corners
Background: Solid green or gradient from green to blue
```

## Checklist de Qualidade

- [ ] Ícones em alta resolução (sem pixelização)
- [ ] Cores consistentes com o tema do app
- [ ] Readable em tamanhos pequenos
- [ ] Padding adequado (não cortar nas bordas)
- [ ] Formato PNG otimizado
- [ ] Fundo apropriado (transparente ou sólido)
- [ ] Testado em diferentes dispositivos
- [ ] Diferenciável do ícone do mecânico

## Arquivo Temporário

Enquanto não houver os ícones finais, você pode usar:
- Cor sólida com inicial "C"
- Emoji âncora (⚓)
- Placeholder gerado online
